HR Attrition Dashboard | Employee Retention Analysis
📌 Project Overview

This project focuses on analyzing employee attrition using an interactive Tableau dashboard. The goal is to identify key factors influencing employee turnover and provide actionable insights to improve retention strategies.

The dashboard enables HR teams to monitor attrition trends, employee demographics, job roles, and satisfaction levels for better workforce management.

🚀 Objectives
Analyze employee attrition patterns
Identify key drivers of employee turnover
Monitor satisfaction and engagement levels
Support data-driven HR decision-making
🛠 Tools & Technologies
Tableau – Data visualization & dashboard creation
Microsoft Excel – Data cleaning and preprocessing
Data Analysis Techniques – KPI tracking, segmentation, trend analysis

📊 Project Workflow
1. Data Collection

Gathered employee dataset including demographics, job details, salary, and satisfaction scores.

2. Data Cleaning
Handled missing values
Standardized categories
Removed inconsistencies
3. Data Transformation
Created calculated metrics (attrition rate, averages)
Grouped data into age bands and categories
4. Dashboard Development

Designed interactive dashboards in Tableau with multiple visual components for analysis.

5. Data Analysis
Identified high attrition in specific roles and departments
Analyzed relationship between satisfaction and attrition
Evaluated workforce trends over time
📊 Key Insights
📌 Higher attrition observed in specific job roles (e.g., Sales Executive, Laboratory Technician)
📌 Lower satisfaction scores linked with higher attrition
📌 Certain age groups show higher turnover trends
📌 Departments like Sales show comparatively higher attrition
📌 Work-life balance and environment satisfaction play a critical role
